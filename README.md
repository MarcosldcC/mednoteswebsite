# MedNotes Web

Site público da MedNotes — landing, planos, contato, termos e privacidade.

## Desenvolvimento

```bash
pnpm install
pnpm dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
pnpm start
```

## Estrutura

- **Site público:** `/` (home), `/planos`, `/contato`, `/sobre`, `/funcionalidades`, `/termos`, `/privacidade`
- Componentes do site em `components/web/`
- Componentes UI compartilhados em `components/ui/`
